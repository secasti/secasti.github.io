## 💻 Ada Developers Academy Capstone: Morti
**Tech Stack:** React, Python, Flask, PostgreSQL, SQL Alchemy

*Morti is a full-stack "digital legacy" application designed to allow users to create and store farewell messages for loved ones, to be released only upon a verified life event. This project required a high degree of sensitivity regarding data privacy and complex relational logic.*

### Technical Highlights:
* **Full-Stack Architecture:** Developed a responsive **[React frontend](https://github.com/secasti/morti-frontend)** and a robust **[Python/Flask backend](https://github.com/annavanwesep/morti-back-end)**.
* **Relational Data Modeling:** Designed a PostgreSQL schema to manage intricate links between Users, Messages (Text/Audio), and Trusted Persons, enabling the "declaration of death" logic and account linking.
* **Security & Authentication:** Implemented secure user registration and login flows to protect sensitive personal messages.
* **Engineering Best Practices:** Managed a version-controlled workflow with over 180 commits, focusing on dependency compatibility and clean API routing.
